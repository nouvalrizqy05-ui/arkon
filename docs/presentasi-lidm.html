Fitur Unggulan

Inovasi Diferensiator

🧠 IRT Rasch Model 1-PL

Mengukur kemampuan laten mahasiswa (θ) — bukan sekedar persentase skor. Soal adaptif dipilih berdasarkan kesulitan (b) yang mendekati θ mahasiswa. Dosen melihat distribusi θ real-time di dashboard.

🤖 Gemini AI Adaptive

AI memberikan hint adaptif saat quiz berdasarkan θ dan jumlah salah. Dosen mendapat AI-generated analytics summary dengan rekomendasi pengajaran otomatis.

🔬 3D Assembly + WebXR AR

Model GLB high-fidelity untuk merakit PC virtual. Buka di smartphone → mode AR otomatis aktif. Menggantikan lab fisik senilai ratusan juta rupiah.

⚡ CPU Visual Simulator

Tulis kode Assembly (LOAD, STORE, ADD, JUMP, HALT) dan lihat data mengalir real-time melalui pipeline CPU: Fetch → Decode → Execute secara visual.

Security & Quality

Keamanan & Standar Industri

🛡️ Security Measures

  • JWT Authentication + Refresh Token Rotation
  • Socket.io JWT Middleware (no anonymous WS)
  • Helmet CSP + Rate Limiting + bcrypt hashing
  • SQL Injection prevention (field whitelisting)
  • XSS sanitization (email templates & AI output)
  • Path traversal prevention (crypto random filenames)
  • RBAC: role-based access (dosen/mahasiswa)

✅ Code Quality

  • Modular architecture (15+ route modules)
  • Service layer (IRT, N-Gain, Socket, AI)
  • Input validation (centralized validators)
  • Error boundaries (graceful UI fallback)
  • PWA + Offline fallback
  • WCAG 2.1 accessibility (skip-nav, focus ring)
  • Code splitting + lazy loading
